# 🎓 Student Admission Portal

An Android-based Student Admission Portal with Online Entry Test System developed using Kotlin and Firebase.

This application digitalizes the university admission process by allowing students to apply online, upload documents, generate challans, and appear in online entry tests directly from a mobile app.

---

# 🚀 Features

## 👨‍🎓 Student Module

- Student Registration & Login
- Admission Form Submission
- Upload Required Documents
- Challan Generation
- Paid Challan Upload
- Online Entry Test (100 MCQs)
- Auto Result Generation
- View Admission Status

---

## 👨‍💼 Admin Module

- Admin Login
- Student Management
- Challan Verification
- MCQ Management
- Result Management
- Admission Approval/Rejection

---

# 🧾 Challan Management System

After signup, the system automatically generates an entry test fee challan using student details.

### Challan Includes

- Candidate Full Name
- Father Name
- CNIC/B-Form
- Email Address
- Application ID
- Department Name
- Entry Test Fee
- Bank Details
- Due Date

### Workflow

1. Student Signup
2. Challan Generation
3. Fee Payment
4. Paid Challan Upload
5. Admin Verification
6. Entry Test Approval

---

# 📝 Online Entry Test System

### Features

- 100 MCQs
- Timer System
- One Question at a Time
- Auto Submit on Timer End
- Instant Result Generation
- Randomized Questions

### Categories

- English
- Mathematics
- General Knowledge
- Computer Science
- IQ Questions
