# 🎓 Student Admission Portal System

A hybrid Student Admission Portal System consisting of:

- 🌐 Web Portal (React.js)
- 📱 Android Mobile App (Kotlin)
- ⚙️ Backend APIs (Node.js)
- 🗄️ Shared Database Architecture

The system digitalizes the university admission process by allowing students to register, apply for admission, upload documents, submit fee challans, and appear in online entry tests through both web and mobile platforms.

---

# 📌 Core Functionalities

## 1. Candidate Registration & Login

Students can create accounts and securely login using email authentication.

---

## 2. Email Verification

Candidates verify their email before accessing admission services.

---

## 3. Admission Form Submission

Students provide:

- Personal Information
- Educational Details
- Department Selection
- Required Documents

---

## 4. Challan Generation

The system automatically generates an entry test fee challan containing:

- Candidate Name
- Father Name
- CNIC/B-Form
- Application ID
- Department
- Fee Details
- Due Date

---

## 5. Challan Upload & Verification

Students upload payment proof after fee submission.

Admins verify payment status before test approval.

---

## 6. Online Entry Test

- 100 MCQs
- Timer System
- Auto Submission
- Randomized Questions
- Instant Evaluation

---

## 7. Result Generation

The system automatically calculates:

- Obtained Marks
- Percentage
- Pass/Fail Status

---

## 8. Administrative Dashboard

Admins can:

- Manage Candidates
- Verify Challans
- Manage MCQs
- Publish Results
- Approve Admissions
